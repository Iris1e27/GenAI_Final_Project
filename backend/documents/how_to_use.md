# Welcome to ResearchHive

Here is an instruction about how to use this app.

## 1. Upload papers

Click 'Upload Paper' in the sidebar to upload your paper (in pdf format). 

## 2. Select only one paper

Then you can click: 
1. Read Paper
2. Generate Summary

## 3. Select more than one paper

Then you can click: 
1. Generate Chart
2. Compare Papers
